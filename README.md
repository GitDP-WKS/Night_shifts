# Night Shift Analyzer — Карточки (Streamlit)

Красивое и понятное Streamlit-приложение для анализа ночной смены в колл-центре.
Интерфейс на русском, карточки настроек, интерактивные графики (Plotly), heatmap и экспорт отчёта.

## Файлы
- `streamlit_app.py` — основное приложение.
- `requirements.txt` — зависимости.
- `README.md` — инструкции (этот файл).
